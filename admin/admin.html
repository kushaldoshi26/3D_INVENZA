XAMPP Setup Guide
  1. Install XAMPP from apachefriends.org
  2. Start Apache + MySQL in XAMPP Control Panel
  3. Copy project to C:\xampp\htdocs\3d-invenza\
  4. Open http://localhost/phpmyadmin → SQL tab → Run setup_all.sql
  5. Open http://localhost/3d-invenza/auth/login.php
  6. Login with: admin@3dinvenza.com / Admin@2025
